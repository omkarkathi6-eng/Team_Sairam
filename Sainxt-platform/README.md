# Sainxt Platform Frontend

## Overview
This is the frontend Next.js application for the Sainxt Platform. It provides the user interface for the individual dashboard and career building tools.

## Prerequisites
- Node.js (v18 or higher recommended)
- npm or pnpm

## Installation
1. Install the dependencies via npm:
   ```bash
   npm install --legacy-peer-deps
   ```

## Running the Application
1. Start the development server:
   ```bash
   npm run dev
   ```
2. Open [http://localhost:3000](http://localhost:3000) in your browser.
